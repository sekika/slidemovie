**slidemovie** は、シンプルなテキストドキュメントとプロフェッショナルな動画プレゼンテーションの橋渡しをする強力な Python ツールです。

**Markdown** の構造的な単純さ、**Microsoft PowerPoint** のデザイン機能、そして **AI 音声合成 (TTS)** の力を組み合わせることで、動画編集ソフトや録音機材を一切使わずに、高品質なナレーション付き動画を迅速に作成できます。

## 仕組み

自動化と人間のデザインセンスの相乗効果を生み出します。

1.  **構成と台本 (Markdown)**: *何を* 話し、表示するかをシンプルなテキストファイルで定義します。
2.  **ビジュアルデザイン (PowerPoint)**: PowerPoint の AI「デザイナー」機能を使って、プロフェッショナルな見た目に整えます。
3.  **制作 (slidemovie)**: ツールが自動的に音声を生成し、スライドを画像化し、すべてを同期させて動画に結合します。

## 主な機能

*   **Markdown ベース**: プログラムコードと同じように、動画の構成をバージョン管理できます。
*   **AI ナレーション**: Google Gemini や OpenAI を活用し、自然な音声解説を生成します。
*   **動画編集不要**: ナレーションの長さに合わせて動画のタイミングが自動調整されます。
*   **差分ビルド**: あるスライドの台本を一行変えただけでも、変更箇所だけを再生成するため、時間と API コストを節約できます。
